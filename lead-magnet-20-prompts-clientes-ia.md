## 🔍 FIND: Discover Opportunities

### 1. Client Prospecting Engine
```
I offer [SERVICE] to [TARGET MARKET]. Find 20 potential clients who:
- Are actively looking for [SERVICE] or showing buying signals
- Have the budget and authority to hire
- Are in [GEOGRAPHIC REGION or INDUSTRY]
For each prospect, provide: company name, why they're a good fit, recent trigger event (funding, expansion, rebrand, etc.), and the best way to reach them.
```

### 2. Competitor Client Mining
```
Analyze the client lists of these competitors: [COMPETITOR 1], [COMPETITOR 2], [COMPETITOR 3]. Identify:
- Clients who might be unhappy (based on reviews, complaints, social media)
- Clients in industries my competitors underserve
- Clients who recently had a change (new CEO, merger, rebrand)
- The best angle to approach each one
```

### 3. Job Posting Goldmine
```
I'm a [SERVICE PROVIDER]. Analyze these types of job postings: [JOB BOARD URLS or DESCRIPTIONS]. Identify:
- Companies hiring for roles I can fulfill as a freelancer/agency
- Budget signals in the posting
- Pain points they're trying to solve
- A tailored pitch for each opportunity
```

### 4. Market Gap Finder
```
Research the [INDUSTRY] market in [REGION]. Find:
- Services that clients need but few providers offer
- Underserved niches with willingness to pay
- Recent regulatory changes creating new demand
- Emerging trends that will drive hiring in the next 6-12 months
- My best positioning opportunity as a [SERVICE PROVIDER]
```

### 5. LinkedIn Lead Generator
```
I help [TARGET CLIENT] achieve [RESULT]. Create a LinkedIn search strategy:
- 10 boolean search strings to find ideal prospects
- 5 InMail subject lines with high open rates
- A 3-touch outreach sequence (connection → value → ask)
- Profile optimization tips to attract inbound leads
```

## 📝 PITCH: Craft Winning Proposals

### 6. Proposal Generator
```
Create a client proposal for:
- Client: [COMPANY NAME]
- Service: [WHAT YOU OFFER]
- Their problem: [PAIN POINT YOU DISCOVERED]
- Budget range: [IF KNOWN]

Include: executive summary, proposed solution, timeline, deliverables, pricing options (3 tiers), case study placeholder, and CTA.
```

### 7. Cold Email Sequence
```
Write a 5-email cold outreach sequence for:
- Recipient: [ROLE] at [TYPE OF COMPANY]
- Service: [WHAT YOU OFFER]
- Unique angle: [YOUR DIFFERENTIATOR]

Email 1: Pattern interrupt + value
Email 2: Social proof + case study
Email 3: Educational content + soft pitch
Email 4: Direct offer + urgency
Email 5: Breakup email + last chance

Keep each email under 150 words.
```

### 8. Discovery Call Script
```
Create a discovery call script for selling [SERVICE] to [TARGET CLIENT]. Include:
- Opening rapport (2 min)
- 7 key questions to uncover pain points
- How to quantify their problem in dollars
- Transition to pitch
- Handling the top 5 objections
- Closing for next steps
- Follow-up email template
```

### 9. Case Study Builder
```
I helped [CLIENT TYPE] achieve [RESULT]. Turn this into a compelling case study:
- Situation: [CONTEXT]
- Challenge: [PROBLEM]
- Solution: [WHAT YOU DID]
- Results: [METRICS — use realistic estimates if needed]

Format it for: (a) website, (b) proposal, (c) social media post
```

### 10. Pricing Psychology
```
I offer [SERVICE] and currently charge [PRICE]. Analyze:
- Is this priced too low, too high, or right for [MARKET]?
- 3 pricing strategies I could use (value-based, tiered, retainer)
- How to present pricing so it feels like a no-brainer
- Anchoring techniques to make my price feel smaller
- How to handle "you're too expensive" objections
```

## 🤖 AUTOMATE: Scale Your Outreach

### 11. n8n Lead Scraping Workflow
```
Design an n8n workflow that:
- Scrapes [SOURCE: LinkedIn, job board, directory] for leads matching [CRITERIA]
- Enriches each lead with company data
- Scores leads by fit (1-10)
- Adds them to a Google Sheet
- Sends me a daily digest email with top 5 leads

Provide the complete n8n JSON workflow.
```

### 12. AI Follow-Up System
```
Create a system prompt for an AI agent that:
- Monitors my email for client replies
- Categorizes replies (interested, not ready, objection, referral)
- Drafts appropriate follow-up responses
- Schedules follow-ups based on interest level
- Escalates hot leads to me immediately
```

### 13. Content-to-Lead Pipeline
```
Design a content strategy that generates leads for [SERVICE]:
- 4 blog post ideas that attract ideal clients
- 4 social media post templates
- A lead magnet idea (free tool/template/audit)
- Email capture → nurture sequence (3 emails)
- How to track which content generates actual leads
```

### 14. CRM Automation
```
I use [CRM TOOL or Google Sheets]. Create an automation that:
- Tracks all client interactions
- Sends reminders for follow-ups
- Flags deals going cold (>7 days no contact)
- Calculates pipeline value
- Generates a weekly activity report
```

### 15. Review & Testimonial System
```
Create an automated system to collect testimonials:
- When to ask (trigger: project completion, milestone, win)
- Email template for asking
- Follow-up sequence if no response
- How to use testimonials across platforms
- Permission and formatting templates
```

## 🧠 CLOSE: Seal the Deal

### 16. Objection Crusher
```
I'm selling [SERVICE] to [TARGET CLIENT]. I keep hearing these objections:
1. "[OBJECTION 1]"
2. "[OBJECTION 2]"
3. "[OBJECTION 3]"

For each objection, provide:
- Why they're really saying this (root cause)
- A response that reframes the conversation
- A question to ask back
- A proof point or example
```

### 17. Urgency Creator
```
I need to close [DEAL VALUE] with [CLIENT NAME] this month. Create:
- 3 legitimate urgency factors I can mention
- A limited-time offer that feels exclusive
- A "walk away" price that protects my margins
- The exact script for the closing conversation
- A deadline email template
```

### 18. Referral Engine
```
Design a referral system for my [SERVICE] business:
- When and how to ask for referrals
- Incentive structure (discount, bonus, reciprocal)
- Email template for referral requests
- How to make it easy for clients to refer (one-click)
- Tracking system for referral sources
```

### 19. Upsell & Cross-Sell
```
My client just purchased [SERVICE A]. Create:
- 3 natural upsell opportunities
- 3 cross-sell services that complement their purchase
- The right timing to pitch each
- Email templates for each upsell
- How to frame it as "more value" not "more cost"
```

### 20. Client Retention System
```
I want to reduce churn and keep clients longer. Design:
- A 30/60/90 day check-in sequence
- Early warning signs a client is leaving
- A "save" script for at-risk clients
- Value-add touchpoints between projects
- Quarterly business review template
```
